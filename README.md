# solr-package-template

A cookiecutter template to build packages in Apache Solr.

To generate a new project,

    pip install cookiecutter

    cookiecutter https://github.com/searchscale/solr-package-template

    # Answer all the questions

    # Then cd into the new project, and read README.md for generating keypair and building instructions.


## Features

* RequestHandlers
* UpdateRequestProcessors
